# FirstBot
A Very Simple Bot That you allow to write in Embeds for example News etc.


The Commands are
.template
.news

#The config.json

{ 
    "token": "YOURTOKEN", //Your Bot token
    "news": "AAAA", // The output channel
    "prefix": ".", //Prefix
    "role": "NEWS" // Only this Role can execute the commands
}
